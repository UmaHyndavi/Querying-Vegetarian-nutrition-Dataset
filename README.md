# Objective:
The objective of this project is to emphasize the importance of vegetarian nutrition and analyze how plant-based foods can offer competitive levels of protein, fiber, vitamins, and minerals when compared with many non-vegetarian alternatives.

## Dataset:

1.Fermented Foods,
2. Fruits,
3.Grains and Millets,
4.Leafy Greens,
5.Nuts and Seeds,
6.Pulses and legumes,

### ## Conclusion

This project demonstrates the use of SQL for nutritional data analysis by building and exploring a comprehensive Vegetarian Nutrition Dataset. Through data cleaning, aggregations, correlated subqueries, CTEs, and window functions, I analyzed nutrient patterns across different food categories.

The analysis revealed that Pulses & Legumes are the strongest plant-based protein source, Nuts & Seeds excel in fiber and healthy fats, Leafy Greens provide the highest average calcium, and Fruits and Vegetables are the richest sources of Vitamin C. Grains & Millets emerged as the primary carbohydrate and energy source.

Overall, this project strengthened my practical skills in SQL, data cleaning, exploratory data analysis, and data modeling, while also producing a dataset that can be extended further with Power BI visualizations, nutritional scoring, and recommendation systems.
